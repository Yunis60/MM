2.2 Filter outlier genes

This step is to filter outlier genes which are often not informative in distinguishing cell types and can be a source of large spurious variance.

input=sc.dat, #make sure the colnames are gene symbol or ENSMEBL ID

2.3 Construct Prism object

This step is to construct a Prism object. The reference parameter is scRNA_seq, while mixture refers to bulk samples. There are two options for parameter input.type: “count.matrix” and “GEP”(Gene Expression Profile). “GEP” is used when using reference derived from other assays, such as sorted bulk data.

myPrism <- new.prism(
  reference=sc.dat.filtered.pc, 
  mixture=bk.dat,
  input.type="count.matrix", 
  cell.type.labels = cell.type.labels[1:5000], 
  cell.state.labels = cell.state.labels[1:5000],
  key="tumor",
  outlier.cut=0.01,
    outlier.fraction=0.1,
)
