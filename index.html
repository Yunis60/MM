1. Brief introduction to BayesPrism

Bayesian cell Proportion Reconstruction Inferred using Statistical Marginalization (BayesPrism): A Fully Bayesian Inference of Tumor Microenvironment composition and gene expression

BayesPrism consists of the deconvolution module and the embedding learning module.

BayesPrism performs cell type and gene expression deconvolution for bulk RNA-seq using scRNA-seq of samples collected from matched or similar tissue type. It treats the scRNA-seq as prior information, and estimates the joint posterior distribution of cell type fraction and cell type-specific gene expression in each bulk conditional on the reference and each observed bulk.

Since the deconvolution module produces cell type proportions, we mainly concern this part. The following is only about the part of BayesPrism model without the embedding learning.

The input of the model are:

  • bulk samples (# of samples by # of genes)
  • scRNA_seq (# of cells by # of genes)
  • cell type labels (# of cell types equal to # of cells in scRNA_seq)
  • cell state labels (# of subtypes)

The output of the model are:

  • Expression matrix of cell types for each sample
  • Initial cell type fractions for each sample
  • prosterior cell type fracions for each sample
